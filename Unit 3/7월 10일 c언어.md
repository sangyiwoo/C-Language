## 반복문
* for
* while
* do while

```c
#include <stdio.h>

int main()
{
    for(int i=0;i<10;++i)
    {
        printf("%d",i);
    }
    int Count;
    while(1)
    {
        printf("%d", Count);
        ++Count;
        if(Count>=10)
        {
            break;
        }
    }
    do while(Count<10) /*안에 있는 코드를 한번정도는
    실행해줘야할때 사용*/
    
    do{
        printf("실행");
    }while(Count<20);
}
```

## EOF
* C언어는 운영체제와 상관없이, 파일의 끝에 도달했을 때 언제나 특별한 값을 반환하도록 합니다.

* 그 값을 EOF(End Of File)라고 하며, 실제로 이 값은 -1을 나타냅니다.
